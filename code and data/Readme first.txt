"mrss.R" provides the main code for the methodology.

"simulation" folder contains all codes for the simulation study:
	"simu.R" is the main script
	"simu_sample_data" folder contains simulation data
	"simu_results" folder contains all simulation results for generating figures. 
	
"real data" folder contains all codes for the real application study:
	"1 download mPower Data.R" is used to download all data and extract features from the data
	"2 build model.R" is used to run model and plot figures
	"mpower_data" folder contains all downloaded data and extracted features
	"extract walking features" folder contains functions for feature extraction for walking modality
	"extract voice features" folder contains functions for feature extraction for voice modality
